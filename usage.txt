Usage example:

-h --help
Brings this screen you are now reading

-method --method=name
Supported names : [native, urllib2]

-v --verbose
Shows diagnostic messages

-d --delimiter
Adds the delimiter definition line to the result CSV

-i --input
Define the filename from which the static data will be read

-s --sites
Define the monitor names from which you want to crawl,
i.e.: --sites=popularhyip or --sites=goldpoll,popularhyip
